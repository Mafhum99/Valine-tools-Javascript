# Activation Rate Calculator

> **Category**: Calculator
> **Icon**: 🚀
> **Type**: Calculator

## 📝 Description
**How it works:** Calculate the activation rate by dividing activated users by total signups. Enter your signup and activation numbers to instantly see your conversion percentage.

## 🚀 How to Use
1. Open `index.html` in your browser
2. Enter the total number of signups
3. Enter the number of activated users
4. View results instantly
5. Copy results to clipboard
