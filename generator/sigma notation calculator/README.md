# Sigma Notation Calculator

Calculate the summation of a function f(n) over a range of integers.

## Features
- Support for mathematical expressions with variable 'n'
- Support for basic operators (+, -, *, /, ^)
- Support for common functions (sin, cos, tan, log, sqrt, abs)
- Constants like pi and e
- Displays total sum, number of terms, and average
- Mobile-friendly design

## Usage
1. Enter the function f(n).
2. Enter the start and end values for n.
3. Click **Calculate Summation**.
