# Abbreviation Expander

> **Category**: Text Tools
> **Icon**: 📖
> **Type**: Other

## 📝 Description
**How it works:** Enter common abbreviations and acronyms to get their full expanded forms. Supports 150+ abbreviations across technology, business, government, and everyday usage. Enter multiple abbrevations separated by spaces or commas for batch expansion.

## 🚀 How to Use
1. Open `index.html` in your browser
2. Enter an abbreviation or acronym (e.g., API, HTML, ROI)
3. Click Expand button
4. View results instantly
5. Copy results if needed
