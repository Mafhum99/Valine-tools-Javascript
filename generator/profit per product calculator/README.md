# Profit Per Product Calculator

> **Category**: Ecommerce Tools
> **Project Number**: 105
> **Folder**: `105_profit-per-product-calculator`

## How it works

Enter selling price and cost price. Shows profit per unit and profit margin percentage.

## How to Use

1. Open `index.html` in your browser
2. Enter the required parameters
3. Click **Calculate** button
4. View results instantly
5. Use **Copy Result** to copy results to clipboard

## Features

- Real-time calculations
- Mobile responsive design
- Copy results to clipboard
- Keyboard shortcuts (Enter to calculate)
- Error handling for invalid inputs
- Clean, modern UI

## Technical Details

- **HTML5** - Semantic markup
- **CSS3** - Responsive design
- **JavaScript (ES6+)** - No dependencies
- **Standalone** - All assets self-contained (no shared.css/js required)

## Troubleshooting

If you encounter issues:
1. Check browser console (F12) for errors
2. Ensure JavaScript is enabled
3. Try clearing browser cache
4. Verify all files are present in folder
