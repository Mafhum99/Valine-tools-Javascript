> **Category**: Generator

## 📝 Description
**How it works:** Generate interactive accordion components from plain text input. Enter title|content pairs, customize header color and style, and get ready-to-use HTML code.

## 🚀 How to Use
1. Open `index.html` in your browser
2. Enter accordion items (one per line in format: title|content)
3. Customize the header color and style (Bordered, Card, or Minimal)
4. View results instantly in the preview and HTML output
5. Copy results to clipboard
