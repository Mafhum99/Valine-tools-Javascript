# Absolute Value Calculator

> **Category**: Math Tools
> **Icon**: |x|
> **Type**: Calculator

## 📝 Description
**How it works:** Calculate the absolute value of any number. Enter a positive, negative, or zero value and the calculator returns its distance from zero on the number line, along with a helpful explanation of the result.

## 🚀 How to Use
1. Open `index.html` in your browser
2. Enter any number (positive, negative, or zero)
3. Click Calculate button
4. View results instantly
5. Copy results if needed
