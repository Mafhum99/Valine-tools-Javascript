# Acronym Generator

> **Category**: Generator
> **Icon**: 🔤
> **Type**: Generator

## 📝 Description
**How it works:** Generate acronyms from phrases by extracting the first letter of each word. Simply enter a phrase and instantly get its corresponding acronym.

## 🚀 How to Use
1. Open `index.html` in your browser
2. Enter a phrase or sentence
3. Click Calculate or press Enter
4. View results instantly
5. Copy results to clipboard
